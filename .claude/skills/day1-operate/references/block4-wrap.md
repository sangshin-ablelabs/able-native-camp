# Block 4: Day 1 마무리

## EXPLAIN

Day 1에서 배운 거야:

```
1. Claude Code = 터미널(또는 VS Code)에서 실행하는 AI (대화 + 실행)
2. CLAUDE.md = AI에게 주는 내 사용설명서
3. 4가지 도구 = Read, Write, Edit, Bash
4. 자연어로 시키면 돼
5. 모르면 Claude에게 물어보면 돼
```

### 오늘의 미션

Teams #able-native-camp 채널에 아래를 공유해:

```
[Day 1 완료]
- 만든 것: CLAUDE.md
- 해본 업무: [실제로 해본 업무 1개]
- 소감: [한줄]
```

