# Block 3: 실제 업무 1개 해보기

## EXPLAIN

이제 연습이 아니라 진짜 업무를 해본다.

아래 중에서 **지금 당장 해야 하는 업무 1개**를 골라서 Claude에게 시켜본다.

### 업무 예시 (직군별)

**누구나:**
- 이메일 답장 초안 쓰기
- 미팅 회의록 정리
- 보고서/문서 초안 작성
- 자료 번역 (한→영, 영→한)
- 데이터 정리/요약

**제조/생산:**
- 생산 일보 양식 초안
- 부품 리스트 정리
- 검사 체크리스트 초안

**연구/개발:**
- 실험 프로토콜 요약
- 논문/특허 자료 정리
- 기술 스펙 비교표

**경영지원/재무:**
- 경비 보고서 정리
- 계약서 조항 요약
- 일정 정리

### 시키는 법

그냥 자연어로 말하면 된다:

```
어제 [미팅명] 미팅에서 논의한 내용을 회의록으로 정리해줘.
참석자는 [A, B, C]이고, 주요 안건은 [...]이야.
```

```
이 메일에 대한 답장 초안 써줘. 톤은 정중하게.
[메일 내용 붙여넣기]
```

```
아래 데이터를 표로 정리해줘.
[데이터 붙여넣기]
```

## EXECUTE

1. 지금 해야 하는 업무 1개를 고른다
2. Claude에게 자연어로 시킨다
3. 결과물을 확인한다
4. 수정할 부분이 있으면 추가 지시한다
5. 최종 결과물을 파일로 저장한다

**중요:** 처음부터 완벽하지 않아도 된다. "이 부분 수정해줘", "톤을 바꿔줘" 등 대화로 다듬으면 된다.
