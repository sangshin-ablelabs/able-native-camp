# Block 2: 기본 도구 4가지

## EXPLAIN

Claude Code에는 핵심 도구 4가지가 있다. 이것만 알면 대부분의 업무를 시킬 수 있다.

### 1. Read — 파일 읽기
```
이 파일 읽어줘: /path/to/file.md
```
어떤 파일이든 읽을 수 있다. 엑셀, PDF, 이미지도 된다.

### 2. Write — 파일 쓰기
```
회의록 파일을 만들어줘
```
새 파일을 만들거나, 기존 파일을 완전히 다시 쓴다.

### 3. Edit — 파일 수정
```
3번째 줄의 "김철수"를 "박영희"로 바꿔줘
```
파일의 특정 부분만 수정한다. 전체를 다시 쓰는 게 아니라 딱 그 부분만.

### 4. Bash — 명령어 실행
```
이 폴더의 파일 목록을 보여줘
```
터미널 명령어를 실행한다. 파일 정리, 프로그램 실행, 뭐든 가능하다.

### 비유

| 도구 | 비유 |
|------|------|
| Read | 서류를 꺼내서 읽는 것 |
| Write | 새 서류를 작성하는 것 |
| Edit | 서류의 오타를 수정하는 것 |
| Bash | 복사기 돌리기, 프린트하기 등 사무기기 사용 |

### 핵심

직접 명령어를 외울 필요 없다. 자연어로 말하면 Claude가 알아서 적절한 도구를 쓴다.
"이 파일 읽어줘" → Claude가 Read 도구를 쓴다
"파일 만들어줘" → Claude가 Write 도구를 쓴다

## EXECUTE

아래 4가지를 순서대로 해본다:

1. **Read**: "이 폴더에 어떤 파일이 있는지 보여줘"
2. **Write**: "오늘 할 일 목록을 todo.md 파일로 만들어줘"
3. **Edit**: "todo.md에서 첫 번째 항목을 완료 표시해줘"
4. **Bash**: "현재 날짜와 시간을 알려줘"
