# Block 0: 첫 대화

## EXPLAIN

Claude Code는 터미널에서 실행하는 AI 도구다. 채팅창이 아니라 터미널이다.

왜 터미널인가?
- 파일을 직접 읽고 쓸 수 있다
- 명령어를 실행할 수 있다
- 내 컴퓨터의 모든 도구에 접근할 수 있다

채팅 AI는 "대화만" 한다. Claude Code는 "대화하면서 실행"한다.

### 실행 방법 — 2가지

**방법 1: 터미널**
```bash
claude
```
이것만 치면 된다. Claude가 인사한다.

**방법 2: VS Code**
VS Code를 쓰고 있다면 Claude Code 확장을 설치하면 에디터 안에서 바로 쓸 수 있다.
- 사이드바 Claude 아이콘 클릭 → 채팅 패널 열림
- 파일을 눈으로 보면서 AI와 대화할 수 있어서 더 편하다
- 코드를 드래그해서 선택 → 채팅에 자동으로 컨텍스트 전달

> 둘 다 같은 Claude Code다. 터미널이든 VS Code든, 편한 걸로 쓰면 된다.

### 첫 대화 예시

아무거나 시켜봐:
- "안녕, 나는 [이름]이야. ABLE Labs [팀]에서 일해"
- "오늘 날짜 알려줘"
- "이 폴더에 어떤 파일이 있는지 보여줘"

### 핵심 마인드셋

**모르면 Claude에게 물어봐.** 기능을 외울 필요 없다. "이거 어떻게 해?" 하면 Claude가 알려준다. 이 캠프 내내 이 원칙이 적용된다.

## EXECUTE

1. 터미널 또는 VS Code에서 Claude Code를 연다
2. `claude`를 입력한다 (VS Code는 사이드바 아이콘 클릭)
3. 아무 말이나 걸어본다 — 자기소개, 간단한 질문, 뭐든 좋다
4. 3번 이상 대화를 주고받아본다
5. (선택) "Claude Code에서 MCP가 뭐야?" 같은 질문도 해본다 — Claude가 자기 기능도 설명해준다
