# Block 0: 스킬 vs 하네스 vs 에이전트

## EXPLAIN

Day 1~4에서 배운 것들을 정리하고, 오늘 배울 "에이전트"가 왜 다른지 이해해봐.

### 4단계 구분

| 단계 | 비유 | 지시 방식 | 예시 |
|------|------|-----------|------|
| **스킬** | "이거 해줘" | 단일 작업 지시 | "이 메일 요약해줘" |
| **하네스** | "이 순서로 해줘" | 스킬 연결 | "메일 확인 → 요약 → 노션 저장" |
| **에이전트** | "이 목표를 달성해줘, 방법은 네가 정해" | 판단 + 실행 위임 | "이번 주 중요한 거 놓친 게 없는지 확인해줘" |
| **서브에이전트** | "이 부분은 전문가한테 시켜" | 에이전트가 부리는 전문가 | 에이전트가 메일 분석은 메일 전문 서브에이전트에게 넘김 |

### 핵심 차이

**스킬/하네스:** 사람이 "뭘 할지"와 "어떤 순서로 할지"를 정해. AI는 시키는 대로 실행만 해.

**에이전트:** 사람이 "목표"만 정해. **"어떻게 할지"는 AI가 스스로 판단해.**

### 비유

- 스킬 = "A서류 복사해와" (구체적 지시)
- 하네스 = "A서류 복사하고, B에게 전달하고, 회신 받아와" (순서 지시)
- 에이전트 = "이번 분기 실적 보고서 준비해줘" (목표만 제시)
  - 에이전트가 알아서: 데이터 수집 → 분석 → 양식 선택 → 초안 작성 → 검토 요청

### 에이전트가 판단하는 것들

1. **어떤 도구를 쓸지** — MCP를 쓸까, 파일을 읽을까, 검색할까
2. **어떤 순서로 할지** — A부터 할까, B부터 할까
3. **예외 상황 처리** — 데이터가 없으면 어떻게 할까
4. **결과 품질 판단** — 이 정도면 충분한가, 더 해야 하나

## EXECUTE

1. 자기 업무 3개를 떠올려봐
2. 각각이 "스킬 수준", "하네스 수준", "에이전트 수준" 중 어디에 해당하는지 분류해봐
3. Claude에게 이렇게 말해봐:

```
내 업무 3개를 스킬/하네스/에이전트 수준으로 분류해줘.

1. [업무 A]
2. [업무 B]
3. [업무 C]

각각 왜 그 수준인지도 설명해줘.
```

---
위 내용을 직접 실행해봐.
실행이 끝나면 "완료" 또는 "다음"이라고 입력해.
