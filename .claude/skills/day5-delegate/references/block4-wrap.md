# Block 4: Day 5 마무리

## EXPLAIN

Day 5에서 배운 거:

```
1. 스킬 = "이거 해줘" / 하네스 = "이 순서로 해줘" / 에이전트 = "이 목표 달성해줘"
2. 에이전트는 목표만 받으면 방법을 스스로 판단해
3. 위임 가능 업무: 판단 여지 있고, 복구 가능하고, 목표가 명확한 것
4. 인스트럭션 4요소: 목표, 사용 가능 도구, 판단 기준, 제약 조건
5. 에이전트는 초안을 만들고, 최종 결정은 사람이 해
```
