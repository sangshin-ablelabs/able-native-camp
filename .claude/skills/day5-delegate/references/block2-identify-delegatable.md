# Block 2: 위임 가능한 업무 찾기

## EXPLAIN

모든 업무를 에이전트에게 위임할 수 있는 건 아니야. 위임에 적합한 업무와 아닌 업무를 구분해야 해.

### 위임 가능 체크리스트

아래 조건을 **2개 이상** 충족하면 에이전트에게 위임할 수 있어:

```
✅ 정답이 여러 개일 수 있어 (판단 여지가 있어)
✅ 데이터 수집 + 정리 + 판단이 섞여 있어
✅ 매번 약간씩 달라 (완전 동일 반복이 아니야)
✅ 목표는 명확하지만 과정이 유동적이야
✅ 실수해도 복구 가능해 (되돌릴 수 있어)
```

### 위임하면 안 되는 것

```
❌ 최종 의사결정 (승인, 계약, 인사)
❌ 민감한 개인정보/기밀 정보 처리
❌ 되돌릴 수 없는 작업 (발송, 삭제, 배포)
❌ 감정적 판단이 필요한 것 (갈등 중재, 위로)
```

**원칙: 에이전트는 초안을 만들고, 최종 결정은 사람이 해.**

### 직군별 위임 가능 업무 예시

**누구나:**
- 주간 업무 정리 및 보고서 초안
- 미팅 준비 자료 수집 + 요약
- 이메일 분류 + 우선순위 판단 + 답장 초안

**제조/생산:**
- 생산 데이터 이상치 탐지 + 원인 추정
- 부품 발주 리스트 작성 (재고 기반)

**연구/개발:**
- 논문/특허 조사 + 관련성 평가
- 실험 결과 패턴 분석 + 인사이트 도출

**경영지원:**
- 비용 데이터 분석 + 절감 포인트 식별
- 계약서 조항 비교 + 리스크 플래그

### 핵심 질문

> **"내 업무 중 목표만 주면 AI가 알아서 할 수 있는 건 뭘까?"**

## EXECUTE

1. 자기 업무 목록을 5개 이상 적어봐
2. 각각에 위임 가능 체크리스트를 적용해봐
3. Claude에게 이렇게 말해봐:

```
내 업무 목록에서 에이전트에게 위임할 수 있는 것을 골라줘.

1. [업무 A]
2. [업무 B]
3. [업무 C]
4. [업무 D]
5. [업무 E]

각각 위임 가능/불가능 판단과 이유를 알려줘.
위임 가능한 것 중 가장 효과가 클 것 1개를 추천해줘.
```

---
위 내용을 직접 실행해봐.
실행이 끝나면 "완료" 또는 "다음"이라고 입력해.
