# Block 3: 졸업 체크리스트 + 수료식

## EXPLAIN

6일간의 여정을 마무리하는 시간이야.

### 졸업 체크리스트

아래 항목을 모두 달성했는지 확인해봐:

```
□ CLAUDE.md 완성 (Day 1)
   → 내 역할, 업무, 선호 스타일, 회사 맥락이 들어 있는지 확인해봐

□ MCP 3개 이상 연결 (Day 2)
   → 캘린더, 메일, Teams, Notion 등 실제 업무 도구가 연결돼 있는지 봐

□ 스킬 1개 이상 생성 (Day 3)
   → .claude/skills/ 에 내가 만든 스킬 파일이 있는지 확인해

□ 워크플로우(하네스) 1개 이상 구축 (Day 4)
   → 스킬 2개 이상이 연결된 워크플로우가 있는지 봐

□ 에이전트 위임 경험 1회 이상 (Day 5)
   → 목표만 주고 AI가 판단해서 실행한 경험이 있는지 확인해

□ 메모리 저장 (Day 6)
   → 피드백/선호도가 메모리에 저장되어 있는지 봐

□ 루틴 세팅 (Day 6)
   → CLAUDE.md에 일일/주간 루틴이 기록되어 있는지 확인해
```

### 6단계 완성

```
Operate → Connect → Automate → Harness → Delegate → Sustain

Day 0: Why      ✅ 왜 해야 하는지 알게 됐지
Day 1: Operate  ✅ 도구를 쓸 수 있게 됐지
Day 2: Connect  ✅ 외부 서비스를 연결했지
Day 3: Automate ✅ 반복 작업을 스킬로 만들었지
Day 4: Harness  ✅ 스킬을 엮어 워크플로우를 만들었지
Day 5: Delegate ✅ 에이전트에게 위임했지
Day 6: Sustain  ✅ 복리로 성장하는 환경을 만들었지
```

### 리더보드 최종 결과 + 시상

```
Best ABLER      — 종합 점수 1위
Best Skill      — 가장 실용적인 스킬을 만든 사람
Best Workflow   — 가장 효과적인 워크플로우를 만든 사람
Best Delegator  — 에이전트 위임을 가장 잘 활용한 사람
Most Improved   — 가장 크게 성장한 사람
```

### 졸업 후

캠프는 끝나지만, 여정은 계속돼.

```
1. 매일 아침 루틴 실행 — 습관을 유지해
2. 새 스킬을 만들 때마다 Teams에 공유 — 조직이 함께 성장해
3. CLAUDE.md를 계속 업데이트 — AI가 나를 더 잘 알게 돼
4. 메모리를 쌓아 — 복리 효과가 커져
5. 동료에게 가르쳐봐 — 가르치면서 더 깊이 이해하게 돼
```

**ABLE Native Camp의 진짜 목표는 6일 교육이 아니야.
27명의 에이블러가 각자의 AI 업무 환경을 갖추는 거야.
오늘은 끝이 아니라 시작이지.**

## EXECUTE

1. 졸업 체크리스트를 하나씩 확인해봐
2. Claude에게 말해봐:

