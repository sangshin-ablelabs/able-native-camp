# Block 5: Day 3 마무리

## EXPLAIN

Day 3에서 배운 거:

```
1. 스킬 = 자주 하는 요청을 한 단어로 실행하는 거야
2. SKILL.md = AI에게 가르치는 업무 매뉴얼 (마크다운 파일 하나)
3. 구조: 프론트매터 → 목적 → 절차 → 참조/템플릿
4. 핵심: "동료에게 인수인계하듯" 구체적으로 써
5. 교환 테스트로 빠진 부분을 찾아
```

### 지금까지의 여정

```
Day 1: Operate  → 도구를 쓸 수 있어 (Claude Code + CLAUDE.md)
Day 2: Connect  → 회사 시스템을 연결해 (MCP)
Day 3: Automate → 반복 업무를 스킬로 만들어 ← 오늘 여기
```

Day 1에서 도구를 익혔고, Day 2에서 회사 시스템을 연결했고, 오늘 그 위에 **자기만의 자동화**를 얹었어.

### 오늘의 미션

Teams #able-native-camp 채널에 아래를 공유해:

```
[Day 3 완료]
- 만든 스킬: /[스킬명] — [한줄 설명]
- 실제 사용 결과: [스킬을 실제 업무에 1번 써본 결과]
- 소감: [한줄]
```

**미션의 핵심**: 만든 스킬을 **실제 업무에 1번** 사용하고 결과를 공유하는 거야. 연습이 아니라 실전이지.

### 내일 예고

Day 4에서는 **Harness**를 배워. 스킬 하나가 아니라, **스킬들을 엮어서 워크플로우**를 만들어.

예를 들어:
- "캘린더 확인 → 오늘 미팅 정리 → 미팅노트 템플릿 생성" 을 한번에
- "메일 확인 → 긴급건 분류 → 팀즈에 알림" 을 한번에

개별 스킬이 레고 블록이라면, Day 4는 레고를 조립하는 날이야.

```
/day4-harness 로 시작하면 돼.
```
