# Block 2: 내 반복 업무 찾기

## EXPLAIN

스킬을 만들기 전에 먼저 해야 할 일 — **뭘 자동화할지 고르기**.

### 반복 업무의 특징

스킬로 만들기 좋은 업무는 이런 특징이 있다:

```
✅ 스킬로 만들기 좋은 업무
- 매주 또는 매월 반복한다
- 할 때마다 비슷한 패턴을 따른다
- 매번 같은 포맷으로 결과물을 만든다
- 할 때마다 "또 이거야" 싶다

❌ 스킬로 만들기 어려운 업무
- 매번 완전히 다른 판단이 필요하다
- 한 번만 하고 다시 안 한다
- 정해진 절차가 없다
```

### 팀별 반복 업무 예시

이건 예시일 뿐이다. 자기 업무에서 찾아야 한다.

**제조팀**
- 주간 생산 실적 정리
- BOM(자재 명세서) 변경 이력 업데이트
- 장비 점검 체크리스트 작성
- 출하 전 검수 리포트
- 재고 현황 요약

**연구팀**
- 실험 데이터 분석 보고서
- 논문/특허 레퍼런스 정리
- 프로토콜 변경 문서 작성
- 시약 주문 목록 정리
- 실험 노트 요약

**경영지원팀**
- 월말 비용 정산 요약
- 회의록 작성 및 배포
- 근태 현황 정리
- 법인카드 사용 내역 정리
- 계약서 검토 체크리스트

**마케팅팀**
- 뉴스레터 초안 작성
- SNS 포스트 초안
- 전시회 준비 체크리스트
- 고객 레퍼런스 정리
- 웹사이트 콘텐츠 업데이트

**영업팀**
- 미팅 후 팔로업 메일
- 견적서 초안 작성
- 파이프라인 주간 업데이트
- 고객 문의 응대 템플릿
- 주간 세일즈 리포트

### 고르는 기준

5개 중에서 1개를 고를 때 이 기준으로 고른다:

```
1순위: 자주 한다 (매주 이상)
2순위: 패턴이 명확하다 (매번 비슷한 절차)
3순위: 결과물 형식이 정해져 있다 (보고서, 메일, 리스트 등)
```

3개 다 해당하면 최고의 스킬 후보다.

## EXECUTE

### 실습: 매주 반복하는 업무 5개 적기

종이, 메모장, 아무 데나 좋다. 아래를 채운다:

```
나의 반복 업무 목록:
1. ________________________________ (빈도: 매일/매주/매월)
2. ________________________________ (빈도: 매일/매주/매월)
3. ________________________________ (빈도: 매일/매주/매월)
4. ________________________________ (빈도: 매일/매주/매월)
5. ________________________________ (빈도: 매일/매주/매월)
```

5개 중에서 **1개**를 고른다. 기준:
- 가장 자주 하는 것
- 절차가 명확한 것
- "이거 매번 하기 귀찮다"는 생각이 가장 강한 것

고른 업무를 Claude에게 말해봐:

```
나는 [팀]에서 일하는데, 매주 [업무]를 반복해.
이걸 스킬로 만들고 싶어. 어떤 절차로 하는지 같이 정리해줘.
```
