# Block 1: CEO 시연 가이드

## EXPLAIN

CEO가 직접 Claude Code를 라이브로 시연한다. 3가지 장면을 보여준다.

### 시연 1: 데일리 브리핑 (2분)

```
/daily-brief
```

실행하면 AI가 알아서:
1. 다우오피스에서 오늘 일정 조회
2. 최근 메일 확인
3. Teams 멘션 조회
4. 종합해서 "오늘 CEO가 집중해야 할 것" 요약

**포인트:** "매일 아침 30분 걸리던 일이 30초로 줄었다"

### 시연 2: 미팅 준비 (2분)

오늘 예정된 미팅 하나를 골라서:

```
오늘 [미팅명] 미팅 준비해줘
```

AI가 알아서:
1. 캘린더에서 미팅 정보 조회
2. 상대방 관련 메일/팀즈 히스토리 검색
3. 관련 노션 문서 찾기
4. 미팅 브리프 생성

**포인트:** "MCP로 도구가 연결되면, AI가 내 업무 맥락을 안다"

### 시연 3: 스킬 실행 (1분)

아무 스킬이나 하나 실행:

```
/competitor-battlecard Hamilton
```
또는
```
/notable-quoting [고객명]
```

**포인트:** "한번 만든 스킬은 슬래시 하나로 반복 실행. 이걸 각자가 자기 업무에 맞게 만드는 게 이 캠프의 목표다"

## EXECUTE

CEO가 위 3가지를 순서대로 라이브 시연한다.
시연 후 에이블러들의 질문을 받는다.
