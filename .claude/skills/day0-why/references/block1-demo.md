# Block 1: Claude Code가 뭘 할 수 있어?

## EXPLAIN

백문이 불여일견. 실제로 어떤 일을 할 수 있는지 봐봐.

### 예시 1: 메일 초안

```
나: "거래처에 납품 일정 변경 메일 써줘. 3/25 → 4/1, 원자재 수급 지연"

Claude:
→ 메일 제목 + 본문 초안 작성
→ "좀 더 부드럽게" → 바로 수정
→ "영문 버전도" → 바로 번역
```

### 예시 2: 데이터 정리

```
나: "이 CSV 파일 읽어서 월별 매출 요약표 만들어줘"

Claude:
→ 파일 직접 열어서 읽기
→ 분석하고 정리
→ 새 파일로 저장
```

### 예시 3: 업무 도구 연결 (Day 2에서 배워)

```
나: "오늘 메일 중에 답장 안 한 거 있어?"
→ 다우오피스 메일함 직접 확인 → 미회신 목록 정리

나: "이번 주 일정 알려줘"
→ 캘린더 직접 조회 → 일정표 만들어줌

나: "Teams에서 나한테 멘션된 거 확인해줘"
→ Teams 멘션 조회 → 요약
```

### 핵심 차이

ChatGPT는 "답변"을 줘. Claude Code는 내 컴퓨터에서 직접 "일"을 해.
파일 읽기, 쓰기, 터미널 명령 실행, 외부 서비스 접근까지 다 돼.

## EXECUTE

위 예시들을 하나씩 보여주면서, "이런 것도 되는구나" 느낌을 전달해.
질문이 있으면 자유롭게 받아.
