# Block 2: 6단계 레벨 소개

## EXPLAIN

이 캠프는 6단계로 올라간다. 각 단계가 전 단계 위에 쌓인다.

```
Level 6: Sustain    ─── 복리로 성장하는 환경을 만든다
Level 5: Delegate   ─── 에이전트에게 위임한다
Level 4: Harness    ─── 스킬들을 엮어 워크플로우를 만든다
Level 3: Automate   ─── 반복 업무를 스킬로 만든다
Level 2: Connect    ─── 업무 도구와 AI를 연결한다
Level 1: Operate    ─── 도구를 쓸 수 있다
```

### 각 레벨의 핵심 한줄

| Level | 이름 | "AI에게 이렇게 말한다" |
|-------|------|---------------------|
| 1 | Operate | "이 파일 읽어줘" |
| 2 | Connect | "내 캘린더에서 오늘 일정 뽑아줘" |
| 3 | Automate | `/meeting-prep` (슬래시 하나로 실행) |
| 4 | Harness | "캘린더 → 메일 → 팀즈 → 브리프 순서로 해줘" |
| 5 | Delegate | "이번 주 미팅 전부 관리해줘. 방법은 네가 정해" |
| 6 | Sustain | AI가 매주 알아서 학습하고 개선한다 |

### 비유

- Level 1~2: **운전면허 따기** — 핸들, 브레이크, 방향지시등
- Level 3: **자주 가는 길 네비 저장** — 매번 주소 안 쳐도 됨
- Level 4: **네비 + 주유 + 주차를 한번에 연결** — 출발만 누르면 끝
- Level 5: **자율주행** — 목적지만 말하면 알아서 감
- Level 6: **자율주행차가 도로를 학습하며 점점 빨라짐**

### 이 캠프가 끝나면

- CLAUDE.md (AI에게 주는 내 사용설명서)
- MCP 3개+ 연동 (다우오피스, Teams, Notion)
- 내 업무 스킬 1개+
- 워크플로우(하네스) 1개+
- 에이전트 위임 경험 1회+

이게 각 에이블러의 최소 졸업 요건이다.
