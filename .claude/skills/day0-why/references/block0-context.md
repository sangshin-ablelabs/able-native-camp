# Block 0: 왜 이 캠프를 하는가

## EXPLAIN

에이블러에게 아래 배경을 설명한다.

### 세상이 바뀌고 있다

최근 한 회사에서 비개발자 팀원이 Claude Code를 사용해 **1시간 만에 기획부터 배포까지** 독립적으로 제품을 완성했다. 개발자에게 한마디도 묻지 않고.

Sequoia Capital의 Alfred Lin은 이렇게 말했다:
> "상위 5~10% 엔지니어가 작년 대비 3배 많이 shipping하고 있다. 하지만 조율과 커뮤니케이션이 새로운 병목이 됐다."

ARR 3.5억 달러 기업 Lovable에서는 **전 직원이 프로덕션에 코드를 배포**한다. Growth 리더도 직접 앱을 개발한다.

### ABLE Labs는 이미 시작했다

CEO 신상은 지난 1개월간 Claude Code로:
- 매일 아침 데일리 브리핑을 자동화했다 (캘린더 + 메일 + 팀즈 → 노션 투두)
- 120개 이상의 업무 스킬을 만들었다 (견적서, 제안서, 경쟁 분석, 채용, IR 등)
- 다우오피스, Teams, SalesMap, Notion을 AI에 직접 연결했다
- 모든 업무 도구가 하나의 AI 시스템 위에서 돌아가는 환경을 구축했다

이제 에이블러 전원이 이걸 할 수 있게 만드는 거다.

### 이 캠프의 목표

"코딩을 배우자"가 아니다.
**"각자의 업무에서, AI를 동료처럼 쓰는 법을 익히자"**가 목표다.

제조팀은 제조 업무에, 연구팀은 연구 업무에, 경영지원은 경영지원 업무에 — 각자의 맥락에서 Claude Code를 쓴다.
