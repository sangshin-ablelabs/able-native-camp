# Block 0: 왜 이 캠프를 하는 걸까?

## EXPLAIN

### 세상이 바뀌고 있어

ChatGPT한테 가끔 질문하는 거랑은 다른 얘기야. 지금은 AI가 **직접 실행**하는 시대야.

- 파일을 읽고, 쓰고, 수정해
- 이메일, 캘린더, 노션에 직접 접근해
- 터미널에서 명령어도 실행해

채팅 AI는 "답변"만 줘. Claude Code는 내 컴퓨터에서 직접 "일"을 해. 이게 핵심 차이야.

### AI Native가 뭔데?

AI를 "가끔 쓰는 도구"로 두는 게 아니라, **일하는 방식 자체**에 AI가 녹아든 거야.

| 상황 | 지금 | AI Native |
|------|------|-----------|
| 메일 쓸 때 | 처음부터 타이핑 | "거래처에 일정 변경 메일 써줘" → 초안 → 다듬기 |
| 데이터 정리 | 엑셀 수식 만들기 | "이 파일 읽어서 월별로 정리해줘" |
| 회의록 | 미팅 끝나고 한참 정리 | "이 내용으로 회의록 만들어줘" → 액션아이템까지 |
| 뭔가 찾을 때 | 구글링 → 여러 페이지 | "이거 알려줘" → 바로 답변 |

이메일 없이 업무가 안 되듯이, AI 없이 일 안 하는 상태. 그게 AI Native야.

### 이 캠프의 목표

"코딩을 배우자"가 아니야.
**"각자 업무에서, AI를 동료처럼 쓰는 법을 익히자"**가 목표야.

제조팀은 제조 업무에, 연구팀은 연구 업무에, 경영지원은 경영지원 업무에 — 각자 맥락에서 Claude Code를 써보는 거야.
