# Block 3: 설치 확인

## EXPLAIN

Claude Code가 설치되어 있는지 확인한다.

### 확인 방법

터미널을 열고 아래를 입력한다:

```bash
claude --version
```

버전 번호가 나오면 성공.

### 아직 설치 안 했다면

```bash
curl -fsSL https://claude.ai/install.sh | bash
```

설치 후 `claude` 를 입력하면 로그인 안내가 나온다. 회사 계정으로 로그인한다.

### VS Code 사용자

VS Code에서 Claude Code 확장을 설치할 수도 있다. 터미널이 익숙하지 않은 에이블러는 이 방법을 추천한다.

## EXECUTE

1. 터미널에서 `claude --version` 실행
2. 버전이 나오면 완료
3. 안 나오면 위 설치 명령어 실행

---

설치가 끝나면 "완료" 또는 "다음"이라고 입력하세요.
