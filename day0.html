AI Native란? 뭘 할 수 있어? 설치 & 첫 대화

1 AI Native가 뭐야?

▼

ChatGPT한테 가끔 질문하는 거랑은 다른 얘기야.
AI Native는 일하는 방식 자체에 AI가 녹아든 거야. 이메일 없이 업무가 안 되듯이, AI 없이 일 안 하는 상태.

뭐가 달라?

상황지금AI Native
메일 쓸 때처음부터 타이핑"거래처에 일정 변경 메일 써줘" → 초안 나옴 → 다듬기
데이터 정리엑셀 수식 직접 만들기"이 파일 읽어서 월별로 정리해줘"
회의록미팅 끝나고 한참 정리"이 내용으로 회의록 만들어줘" → 액션아이템까지
뭔가 찾을 때구글링 → 여러 페이지 읽기"이거 알려줘" → 바로 답변

핵심은 간단해. 내가 처음부터 다 하지 않고, AI가 80% 만들어준 걸 내가 다듬는 거야.

2 Claude Code는 뭘 할 수 있어?

▼

일반 AI 챗봇이랑 가장 큰 차이 — Claude Code는 내 컴퓨터에서 직접 실행돼.
파일 읽기, 쓰기, 터미널 명령어 실행까지 다 해. 답변만 하는 게 아니라 실제로 일을 해줘.

예를 들면 이런 거야

나:
"이번 주 메일 중에 답장 안 한 거 있어?"

Claude:
→ 다우오피스 메일함 확인 → 미회신 3건 정리해서 보여줌
나:
"이 CSV 파일 읽어서 요약표 만들어줘"

Claude:
→ 파일 직접 열어서 → 분석하고 → 정리된 표를 새 파일로 저장
나:
"거래처에 납품일정 변경 메일 초안 써줘. 3/25 → 4/1, 원자재 수급 지연"

Claude:
→ 메일 초안 작성 → "영문 버전도?" → 바로 번역

ChatGPT는 "답변"을 줘. Claude Code는 내 컴퓨터에서 직접 "일"을 해. 이게 핵심 차이야.

6일 뒤에는 이런 것도 할 수 있어

  • 이메일, 캘린더, Teams, Notion을 Claude에 연결해서 한 번에 확인 (Day 2)
  • 반복 업무를 명령어 하나로 자동화 (Day 3~4)
  • 복잡한 업무를 여러 단계로 쪼개서 한 번에 처리 (Day 5)

3 설치하고 첫 대화해보자

▼
  1. 1
    터미널 열기
    Mac: ⌘ + Space → "터미널" 검색
    Windows: Win + X → "PowerShell" 선택
    VS Code 쓰면 Ctrl + ` 로 내장 터미널 열어도 돼.
  2. 2
    Claude Code 설치
    Mac/Linux: curl -fsSL https://claude.ai/install.sh | bash
    Windows: irm https://claude.ai/install.ps1 | iex
  3. 3
    캠프 저장소 다운로드
    git clone https://github.com/sangshin-ablelabs/able-native-camp.git
    git이 없다면? Mac: xcode-select --install / Windows: git-scm.com에서 설치
  4. 4
    Claude Code 실행
    cd able-native-camp && claude
    처음 실행하면 브라우저가 열려. Anthropic 계정으로 로그인하면 돼.
    계정이 없으면 claude.ai에서 가입. Pro/Max/Teams 구독이 필요해.
  5. 5
    첫 대화
    안녕! 나는 [이름]이야. 잘 부탁해.
    자유롭게 2~3개 질문해봐. 뭐든 괜찮아.

막히면 Teams #able-native-camp 채널에 물어봐. 누군가 바로 도와줄 거야.
